Як-42 для X-Plane 11.

Эта версия основана на Як-42 от Project Oblivion.


Что сделано на данный момент:

1. Параметры двигателей приближены к реальным. Расхождения есть, но для их исправления требуется
построение модели двигателя независимой от X-Plane.

2. Исправлена ситуация с топливными баками. Теперь их три как и положено. Расход топлива не реалистичный.

3. Частично исправлена центровка самолета.


Основные визуальные недостатки:

1. Виртуальной кабины по прежнему нет!

2. Системы самолета не реализованы!

3. Передняя стойка шасси уходит в землю.

4. Выхлоп от двигателей слишком насыщен.
